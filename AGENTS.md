# FlowPay Agent Notes

- Product: Solana-based on-chain invoice cashflow protocol.
- On-chain source of truth: `programs/flowpay`.
- Frontend: `apps/web` with Next.js app router and premium light fintech dashboard styling.
- Metadata layer: Prisma SQLite inside `apps/web/prisma`.
- SDK: `packages/sdk`.
- Demo target: Localnet or Devnet with Mock USDC.
